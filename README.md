# UX Copywriting — Kiro Power

A specialized Kiro Power for UX copywriting that helps craft clear, consistent, and intent-driven copy for user interfaces.

## What it does

- Write UI copy: buttons, labels, tooltips, empty states, error messages, success messages, onboarding flows, modals, banners, notifications
- Review and improve existing copy for clarity, consistency, tone, accessibility, and brevity
- Audit copy across screens and flows for terminology drift, tone shifts, or conflicting messages
- Create copy systems: terminology glossaries, voice and tone guidelines, copy patterns, string tables
- Write microcopy variants for all states: default, loading, empty, error, success, edge cases

## Installation

1. Open Kiro
2. Open the Powers panel
3. Click "Add Custom Power"
4. Select "GitHub Repository"
5. Enter: `https://github.com/HB7918/kiro-ux-writing`
6. Click "Add"

## Reference Guides

This power includes 7 steering files loaded on-demand:

| Steering file | What it covers |
|---|---|
| voice-and-tone | Tone traits, contractions, modal verbs, requirements phrasing |
| alerts-and-messages | Alert types (info, error, success, warning), dos and don'ts |
| error-messages | Error anatomy, validation/quota/permission/internal error patterns |
| ui-copy-guidelines | UI tenets, checklist, before/after examples, capitalization |
| accessibility-and-inclusion | Accessible content, device-agnostic verbs, inclusive language |
| writing-mechanics | Active voice, simple words, fluff removal, jargon, global English |
| conversational-ai | AI identity, response formatting, quality, review checklist |
